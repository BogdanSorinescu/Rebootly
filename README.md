# Rebootly

Rebootly is a web-based platform that helps users troubleshoot, repair, and reuse mobile devices. The application uses an AI assistant to guide users through common phone problems and provides step-by-step software troubleshooting advice.

If the issue cannot be solved through troubleshooting, Rebootly can connect the user with verified repair technicians for further support, repair estimates, or device repair services.

## Purpose

The goal of Rebootly is to make device troubleshooting and repair easier, faster, and more sustainable. Instead of replacing devices too quickly, users are encouraged to repair or reuse existing electronics, helping reduce electronic waste.

## Main Features

- AI-powered troubleshooting chat
- Step-by-step software fix instructions
- Hardware repair referral to verified technicians
- User login and session history
- Appointment booking for repairs
- Secure payment handling
- Refurbished device support
- Admin tools for managing users, experts, reports, and AI data

## Tech Stack

- React
- JavaScript / TypeScript
- PostgreSQL
- OpenRouter for AI model access
- Clerk for authentication
- Stripe for payments
- Vercel for web hosting
- Railway for database hosting

## System Overview

The user interacts with the Rebootly web application through a browser. The frontend and backend logic are handled by the web application, while external services are used for authentication, payments, AI responses, and database storage.

The main flow is:

1. The user describes a device problem.
2. The AI assistant analyzes the issue.
3. The system provides troubleshooting steps.
4. If needed, the user is referred to a technician.
5. The user can book support, send in the device, or receive further repair guidance.

## Project Goal

Rebootly supports a more sustainable approach to technology by promoting repair, refurbishment, and reuse. The platform aims to reduce electronic waste while giving users affordable and reliable repair options.
